## Installation

1. **Clone the repository**:
```bash 
git clone https://github.com/yourusername/finance-tracker.git 
cd finance-tracker
```

2. **Install Required Libraries**:
```
pip install -r requirements.txt
```

3. **Run the program**:
```
python main.py
```
